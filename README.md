## üõ†Ô∏è Installation

### Prerequisites

- Python 3.8 or higher
- Google AI API key (get from [Google AI Studio](https://aistudio.google.com/apikey))

### Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/medical-imaging-diagnosis-agent.git
   cd medical-imaging-diagnosis-agent
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application:**
   ```bash
   streamlit run ai_medical_imaging.py
   ```

4. **Access the app:**
   Open your browser and navigate to `http://localhost:8502`

## üìñ Usage

1. **Configure API Key:**
   - Enter your Google AI API key in the sidebar
   - The key is stored securely in session state

2. **Upload Medical Image:**
   - Click "Upload Medical Image" button
   - Select a supported image file (JPG, JPEG, PNG, DICOM)

3. **Analyze Image:**
   - Click "üîç Analyze Image" button
   - Wait for AI analysis to complete

4. **Review Results:**
   - View structured diagnostic report
   - Includes technical findings, patient explanations, and research references
